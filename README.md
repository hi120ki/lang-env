# Lang Env

An ansible template for language environment by anyenv

## Additional tools

- Devbox

<https://www.jetpack.io/devbox/>

```
curl -fsSL https://get.jetpack.io/devbox | bash -s -- -f
```

- Rye

<https://rye-up.com/>

```
curl -sSf https://rye-up.com/get | RYE_INSTALL_OPTION="--yes" bash
```
